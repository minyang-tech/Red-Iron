# General
Genre : FPS

Game Theme : Berlin, During The Eastern Front

Each round lasts 10 minutes. 

During this time, the primary objective for each faction is to defend or recapture Berlin.

# Mission
Germany Army : Protect Berlin Against the Soviet army

Soviet Army : Recapture Berlin from the German army

# Main System
Players can choose their preferred faction.

Each faction has a 20-player limit. 

If a side is full, you will be automatically assigned to the opposing team.

You Can Run by Press "Shift"

You Can Crouch by Press "Left Ctrl"

You Can Perform a slide by pressing the crouch key while sprinting.

Press [V] for Quick Weapon Swap.

Faction Weapons

Try to capture or defend all 5 strategic points!

# Official Map List
Map 1. Fall of the Reichstag

Feature: Intense CQC (Close Quarters Combat) within the halls and rooms of the parliament. Every corner is a potential ambush.

Map 2. Dead Zone: Brandenburg Gate

Feature: Wide open spaces with minimal cover. A paradise for snipers and tank commanders to dominate from a distance.

Map 3. Fortress: Zoo Flak Tower

Feature: A nightmare for the Soviets, a sanctuary for the Germans. This map features a brutal siege on a nearly impenetrable concrete fortress.

Map 4. The Last Bunker (Chancellery)

Feature: Underground warfare inside the Führerbunker. Focuses on dark, claustrophobic hallway fights and clearing operations.

Map 5. Moltke Bridgehead

Feature: Strategic advantage for the Soviet Army. Utilize the abundant cover along the riverbanks to push through the German lines.

# 공식 맵
Map 1. 의사당의 함락 (Reichstag Annex)

특징: 국회의사당 내부의 좁은 복도와 방 하나하나를 두고 벌이는 숨 막히는 근접전. 샷건과 SMG가 전장을 지배합니다.

Map 2. 사선의 광장 (Brandenburg Gate)

특징: 넓은 개활지와 적은 엄폐물. 저격수의 정밀 사격과 전차의 압도적인 화력이 승부를 결정짓는 장거리 교전 지역입니다.

Map 3. 죽음의 대공포탑 (Zoo Flak Tower)

특징: 독일군의 난공불락 요새. 소련군에게는 지옥 같은 돌격을, 독일군에게는 철벽 방어의 재미를 선사하는 요새전 맵입니다.

Map 4. 지하의 종말 (The Führerbunker)

특징: 총통 관저와 지하 벙커를 배경으로 한 실내 전투. 어두운 통로에서 벌어지는 기습과 처절한 소탕전이 핵심입니다.

Map 5. 몰트케의 진격 (Moltke Bridge)

특징: 소련군에게 유리한 전장. 강변을 따라 배치된 수많은 엄폐물을 활용해 다리를 건너 독일군의 심장부로 진격하세요.
