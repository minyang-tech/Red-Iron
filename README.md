# General
Genre : FPS
Game Theme : Berlin, During The Eastern Front
Each round lasts 10 minutes. 
During this time, the primary objective for each faction is to defend or recapture Berlin.

# Mission
Germany Army : Protect Berlin Against the Soviet army
Soviet Army : Recapture Berlin from the German army

# Main System
Players can choose their preferred faction.
Each faction has a 20-player limit. If a side is full, you will be automatically assigned to the opposing team.

You Can Run by Press "Shift"
You Can Crouch by Press "Left Ctrl"
You Can Perform a slide by pressing the crouch key while sprinting.

Faction Weapons

Soviet Union:
Mosin-Nagant
PPSh-41
DP-27
Heavy Machine Gun (HMG)

German Forces:
Kar98k
MP40
StG44
Gewehr 98

Common (Both Sides):
Looting enemy weapons
Grenades

Try to capture or defend all 5 strategic points!

# 메인 시스템
플레이어는 원하는 진영을 선택할 수 있습니다.
각 진영의 최대 정원은 20명으로, 꽉찰시 자동으로 상대편이 선택됩니다.

Shift를 통해 달립니다.
Ctrl 키를 통해 포복(앉기)이 가능합니다.
달리는 채로 포복 키를 누를시, 슬라이드가 가능합니다.

각 진영에게 부여된 총은 다음과 같습니다.

소련군 :
모신나강
PPSh-41
DP-27
중기관총

독일군 :
Kar98
MP40
StG44
Gewehr 98

공통 :

수류탄
